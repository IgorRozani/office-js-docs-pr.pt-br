# <a name="formsettings-element"></a>Elemento FormSettings

Especifica a localização de origem e as configurações de controle para o seu suplemento de email.

**Tipo de suplemento:** Email

## <a name="syntax"></a>Sintaxe

```XML
<FormSettings>
   ...
</FormSettings>
```
